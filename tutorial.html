PANDUAN INSTALASI
SUNDA NUSWANTARA WALLET

LANGKAH 1
Buka file APK. Saat muncul pertanyaan konfirmasi, tekan tombol INSTAL.
LANGKAH 2
Jika muncul Google Play Protect, pilih Pindai aplikasi untuk memastikan keamanan.
LANGKAH 3
Tunggu sejenak saat sistem memindai keamanan aplikasi.
LANGKAH 4
Setelah terdeteksi aman ("Aplikasi ini tampak aman"), tekan tombol Instal.
LANGKAH 5
Konfirmasi terakhir. Tekan INSTAL sekali lagi untuk menyelesaikan.
LANGKAH 6
Aplikasi berhasil terpasang. Tekan BUKA untuk menjalankan.
LANGKAH 7
Berikan izin akses (Foto/Media) agar fitur QR Code scanner bisa berjalan. Tekan IZINKAN.
LANGKAH 8
Buat PIN Keamanan Baru (6 digit) atau masukkan PIN jika sudah pernah mendaftar.
LANGKAH 9
Selamat! Anda sudah masuk di Dashboard Utama. Saldo dan menu transaksi siap digunakan.
LANGKAH 10
Gunakan menu QR Saya untuk menerima dana atau token dari pengguna lain.
